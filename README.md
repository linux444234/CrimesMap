# CrimesScaner

Pequeno projeto para visualizar crimes em um mapa (Leaflet) e armazenar dados em JSON.

## Como executar

1. Instale dependências:

```bash
npm install
```

2. Para rodar em desenvolvimento (recarregamento automático):

```bash
npm run start:dev
```

3. Para rodar normalmente:

```bash
npm start
```

## Scripts úteis

- `npm run lint` — verifica problemas de estilo com ESLint
- `npm run format` — formata arquivos com Prettier
